Scripts for automating the process of checking out specific Git tags.

## Basic Checkout Script

```bash
#!/bin/bash
# checkout-tag.sh

TAG=$1
if [ -z "$TAG" ]; then
    echo "Usage: $0 <tag>"
    exit 1
fi

git checkout "tags/$TAG"
```
