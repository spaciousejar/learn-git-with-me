# Kanban Boards

GitHub Projects provides Kanban-style boards for visual project management.

## Features

- **Columns**: Organize work into stages (To Do, In Progress, Done)
- **Cards**: Represent issues, pull requests, or notes
- **Automation**: Automatically move cards based on actions
- **Filtering**: View specific subsets of work

## Creating a Board

<Steps>
1. **Navigate to Projects**: Go to your repository or organization
2. **Create project**: Click "New project"
3. **Choose template**: Select "Kanban" or start from scratch
4. **Add columns**: Create workflow stages
5. **Add cards**: Link issues and PRs
</Steps>
