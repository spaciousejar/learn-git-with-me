## Add file to staging
#### for single file
```sh
git add <file>
```
#### for all files
```sh
git add .
```
or
```sh
git add -A
```
#### for all files in a directory
```sh
git add <directory>
```
Adds a file or directory to the staging area. This command
prepares the changes for the next commit. It adds the
specified file or directory to the index.
