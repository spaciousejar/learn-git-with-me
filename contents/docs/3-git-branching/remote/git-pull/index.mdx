## Pull changes
    
    ```sh
    git pull
    ```
    for a specific branch
    example:-main branch
    
    ```sh
    git pull origin main
    ```
Updates the local repository with changes from the remote
repository. It pulls the changes from the remote repository
and merges them with the local changes.
